# poc-react-flow

## Quick Start

```bash
# Preparing NVM ENV
nvm install 20.12.1
nvm alias [project-name] 20.12.1
nvm use [project-name]

# Preparing pnpm as package manager
npm install -g corepack
corepack enable

# Installing packages
npm install

# Starting app in development mode
npm run dev

# Starting app in production mode
npm run build
npm -g install serve (run once)
serve -s dist
```
